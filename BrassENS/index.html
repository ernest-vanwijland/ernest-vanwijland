BrassENS

Bienvenue sur la page de l'atelier de distillation clandestin de l'ENS.

Matériel pour distillation :

  • Casserole avec couvercle fermant hermétiquement
  • Grand tuyau, si possible en cuivre
  • Sceau pour refroidissement
  • Petit bol pour récupérer le méthanol
  • Récipient pour récupérer l'alcool
  • Réchaud
  • Thermomètre
  • Alcoomètre

Matériel pour fermentation (recettes plus précises à venir) :

  • Sceau fermant hermétiquement
  • Sucre
  • Eau
  • Levure
